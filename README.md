# Qing Pu Elementary School — English Website

A full redesign of the English website for **Qing Pu Elementary School** (青埔國小),
a bilingual elementary school near THSR Taoyuan Station in Taoyuan, Taiwan.

## 🔗 Live site
[View on GitHub Pages](https://thetmyatnoe97.github.io/Qing-Pu-Elementary-School/)

## 📄 Pages
- **Home** — hero, stats, news, bilingual programs overview
- **About** — school history timeline, vision pillars, building design story
- **Bilingual Education** — curriculum by grade, subject cards, English journal
- **News & Events** — filterable news grid with live search
- **Gallery** — masonry photo grid with lightbox and touch swipe
- **Visit** — Google Maps, transport directions, nearby landmarks
- **Contact** — validated form, PHP mailer, FAQ accordion

## 🛠 Built with
- HTML5, CSS3 (CSS Grid, Flexbox, CSS Variables)
- Vanilla JavaScript (no frameworks or libraries)
- PHP contact form mailer
- Fully responsive — mobile, tablet, desktop

## ✨ Features
- CSS masonry gallery with JS lightbox (keyboard + swipe navigation)
- Live news search and category filter
- Sticky section nav with scroll-spy
- Contact form with client-side validation and AJAX submit
- Accessible — semantic HTML, ARIA labels, skip link, focus management
